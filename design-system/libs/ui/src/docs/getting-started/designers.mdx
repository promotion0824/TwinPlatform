# Getting Started For Designers

How to get started using our components and tokens in Figma.

## Before you begin

Please familiarise yourself with our [design guidelines](?path=/docs/design-system-color-palette--docs).

## Installing font files

We use the Google Fonts typeface Poppins in our UI and components. Before starting work in Figma, please download and install Poppins:

<p><a href="https://fonts.google.com/download?family=Poppins" target="_blank">https://fonts.google.com/download?family=Poppins</a></p>

## Enabling Figma libraries

We maintain the following Figma libraries for designers to use:

- <a href="https://www.figma.com/file/vEbPL0OE6kcP6SzJfjtl2y/WDS-Design-Tokens-Dark" target="_blank">WDS Tokens Dark</a> - our dark theme design tokens maintained as Figma styles
- <a href="https://www.figma.com/file/A8iQxtuHwI8Jj696ROXgnn/WDS-Design-Tokens-Light" target="_blank" >WDS Tokens Light</a> - our light theme design tokens maintained as Figma styles
- <a href="https://www.figma.com/file/qptMyrqn6VuLgIyrjlhzCO/WDS-UI-Kit" target="_blank">WDS UI Kit</a> - our component and patterns library
- <a href="https://www.figma.com/file/SvJBsPrjU6hXfTMTignDEk/WDS-Icons" target="_blank">WDS Icons</a> - our icon library

To get going using our design system, you simply need to enable the <a href="https://www.figma.com/file/vEbPL0OE6kcP6SzJfjtl2y/WDS-Design-Tokens-Dark" target="_blank">WDS Tokens Dark library</a> and the <a href="https://www.figma.com/file/qptMyrqn6VuLgIyrjlhzCO/WDS-UI-Kit" target="_blank">WDS UI Kit</a>.

See the following Figma documentation on enabling libraries:

- <a href="https://help.figma.com/hc/en-us/articles/360041051154-Guide-to-libraries-in-Figma" target="_blank">Enable libraries in drafts, teams, and files</a>
- <a href="https://help.figma.com/hc/en-us/articles/360039234193" target="_blank">Accept updates from a library</a>
