# Style Props

The following style props are available for use on all of our components, except for those listed below. These props should be used to reduce boilerplate when adding adhoc styles to components when needed, and to provide easy access to our theme tokens as needed.

## Unsupported Components

- [DataGrid](?path=/docs/components-datagrid--docs)
- [NavList](?path=/docs/components-navlist--docs)
- [Snackbar](?path=/docs/components-snackbar--docs)
- All chart components except for [ChartCard](?path=/docs/components-charts-chartcard--docs) and [Tracker](?path=/docs/components-charts-tracker--props)

## Example

```tsx
<Card c="intent.notice.fg.default" m="s16" p="s16">
  Example Card
</Card>
```
