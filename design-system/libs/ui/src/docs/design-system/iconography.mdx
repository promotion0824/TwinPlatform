# Iconography

Our `<Icon>` [component](?path=/docs/components-icon--docs) supports Google's [Material Symbols](https://fonts.google.com/icons) icon library.

## Figma Library

To promote consistency we maintains a small subset of icons alongside guidance for their usage in our Figma icons libary. This library can very easily be extended for sensible usecases. See the [Getting Started for Designers](?path=/docs/getting-started-designers--docs#enabling-figma-libraries) for instructions for how to add an icon to the library.
