# MQTT Receiver

```
Description:
  Receives messages from MQTT

Usage:
  Receiver [options]

Options:
  -c, --cert <cert> (REQUIRED)                  The client certificate file
  -p, --passphrase <passphrase>                 The certificate passphrase
  -s, --subscription <subscription> (REQUIRED)  The MQTT subscription ID
  -m, --mqtt <mqtt> (REQUIRED)                  The MQTT URL
  --version                                     Show version information
  -?, -h, --help                                Show help and usage information
```
