# MQTT Sender

```
Description:
  Sends messages to MQTT

Usage:
  Sender [options]

Options:
  -s, --subscription <subscription> (REQUIRED)           The MQTT subscription ID
  -c, --cert <cert> (REQUIRED)                           The client certificate file
  -p, --passphrase <passphrase>                          The certificate passphrase
  -m, --mqtt <mqtt> (REQUIRED)                           The MQTT URL
  -d, --dest, --destination <destination> (REQUIRED)     The destination subscription ID
  -ci, --connectorid, --connid <connectorid> (REQUIRED)  The Connector ID
  -ei, --externalid, --extid <externalid> (REQUIRED)     The External ID
  --version                                              Show version information
  -?, -h, --help                                         Show help and usage information
```
